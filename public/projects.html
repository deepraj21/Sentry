How to create a GitHub token
  1. Open GitHub → Settings → Developer settings → Personal access tokens.
  2. Create a classic token (or fine-grained token scoped to this repo).
  3. For private repos: enable the repo scope.
  4. For public repos with comments: enable public_repo or pull request write access.
  5. Copy the token and paste it here. Sentry encrypts it before storing.
